# Essentials

## Project setup

```
# npm
npm install
```

### Compiles and hot-reloads for development

```
# npm
npm run dev
```

### Compiles and minifies for production

```
# npm
npm run build
```

### Run all unit tests in terminal
```
# npm
npm run test:unit
```

### Run all unit tests on browser
```
# npm
npm run test:unit:dev
```

### Lints and fixes files

```
# npm
npm run lint
```
